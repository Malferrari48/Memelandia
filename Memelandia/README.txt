Per far funzionare correttamente l'applicazione bisogna:
- Scaricare la cartella
- Aprire il terminale e andare nella cartella del progetto
- Digitare "pipenv shell" per creare un ambiente virtuale
- Digitare "pipenv install" per installare tutte le dipendenze
- Digitare "python manage.py runserver" per rendere operativa
	l'applicazione su localhost:8000/
	
Nel caso non fosse installato sulla macchina bisogna scrivere
il comando "pip3 install pipenv"
